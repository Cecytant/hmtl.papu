Proyecto Aula

¿Que es el proyecto aula?

El Proyecto Aula es una propuesta metodológica que permite incorporar las competencias adquiridas por el alumno en diferentes Unidades de Aprendizaje para dar solución a un problema mediante el método de proyectos en los diferentes semestres.









¿Por que elegimos este tema?

Elegimos este tema para hacer conciencia sobre el mal uso del agua que le damos los seres humanos. Tratamos de dar consejos para poder cuidar y darle el trato que neccesia el agua para que no nos quedemso sin ella.
